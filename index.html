❓ How to Use ▶

Navigation:

  • Tap nodes to expand/collapse branches
  • Pinch to zoom in/out
  • Drag to pan around the tree
  • Use "Reset View" to return to default zoom

Node Symbols:

  • + = Node can be expanded
  • − = Node is expanded
  • No symbol = End skill (tap to view jobs)

Finding Jobs:

  • Tap end nodes (skills) to view available jobs
  • Use search to find specific skills
  • Filter by job count using the slider
